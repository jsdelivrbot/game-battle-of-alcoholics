# Ирга "Битва алкоголиков"

## Установка
Клонируем репозиторий и переходим в эту папку, создаем вторую вкладку в консоли

### В первой печатем:
``` bash
cd client
npm install
npm run dev
```
###Во второй печатем:
``` bash
cd server
npm install
node index.js
```

Затем переходим по адресу http://localhost:8080/ и балуемся.
